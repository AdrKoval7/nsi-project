# Реестр нормативно-справочной информации (НСИ)

Этот проект представляет собой полнофункциональное веб-приложение для просмотра и управления справочниками НСИ. Приложение состоит из фронтенда на Vue.js 2 и бэкенда на Node.js с базой данных MongoDB.

## Структура проекта

Проект организован как монорепозиторий и содержит два основных компонента:

-   `/nsi-backend`: Бэкенд на Node.js/Express, который предоставляет API для работы с данными.
-   `/nsi-frontend`: Фронтенд на Vue.js 2, который обеспечивает пользовательский интерфейс.

## Установка и запуск

### Требования
-   Node.js
-   npm
-   MongoDB

### 1. Настройка Бэкенда
```
# Перейдите в папку бэкенда
cd nsi-backend

# Установите зависимости
npm install

# Создайте файл .env в корне nsi-backend и добавьте строку подключения к БД и изменением порта
# MONGO_URI=mongodb://localhost:27017/nsi_registry
# PORT=3001

# Запустите сервер
node server.js
```
Сервер будет доступен по адресу `http://localhost:3001`.

### 2. Настройка Фронтенда

```
# Из корня проекта перейдите в папку фронтенда
cd nsi-frontend

# Установите зависимости
npm install

# Запустите приложение для разработки
npm run serve
```
Приложение будет доступно по адресу `http://localhost:8080`.
